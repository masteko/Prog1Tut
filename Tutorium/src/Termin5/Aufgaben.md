# Aufgaben

## Aufgabe 1

Schreiben Sie eine Methode , die ein Array von int Werten übergeben bekommt und die Summe aller Elemente im Array zurückliefert.
___

## Aufgabe 2

Schreiben Sie eine Methode, die ein Array von int Werten übergeben bekommt und die Anzahl der Geraden und Ungeraden Elemente auf der Konsole ausgibt.

___

## Aufgabe 3

Schreiben Sie eine Methode, die einen String übergeben bekommt und alle Kleinbuchstaben des Strings durch Großbuchstaben, sowie alle
Großbuchstaben durch Kleinbuchstaben ersetzt. Folgende Klassen bzw. Methoden könnten Ihnen dabei helfen:

- Character.isUpperCase
- Character.isLowerCase
- Character.toUpperCase
- Character.toLowerCase

Sowie bereits bekannte Methoden der String Klasse

___

## Aufgabe 4

Schreiben Sie eine Methode, die ein Array von int Werten übergeben bekommt und den MinSort, den Sie aus der Vorlesung kennen, implementiert.
Die Methode soll daraufhin das sortierte Array zurückliefern.
