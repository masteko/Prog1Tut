## Aufgabe: ungerade Zahlen
a) Schreiben Sie eine Methode die eine natürliche Zahl n übergeben
bekommt und, angefangen von 0, alle ungeraden Zahlen bis n
ausgibt

```
Beispiel:
Eingabe: 5
Ausgabe: „1 3 5“
```

b)

## Aufgabe: Primzahlen die Zweite

Schreiben Sie das Primzahlen-Programm aus Termin 1 so um, dass die Prüfung ob es sich um eine Primzahl handelt in einer eigenen Methode durchgeführt wird. Die Methode liefert als Rückgabeparameter einen boolean, der angibt ob es sich um eine Primzahl handelt oder nicht. Rufen Sie diese Methode an geeigneter Stelle in Ihrem Programm auf.
