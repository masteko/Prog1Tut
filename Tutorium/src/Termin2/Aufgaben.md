# Aufgaben

## Aufgabe: Wochentag
Schreiben Sie eine Methode, die einen Wochentag als String
entgegen nimmt und den Tag der Woche als Zahl zwischen 0 bis 6
zurückliefert.
Bei ungültigem Wert soll die Methode 1 zurückgeben.

___

## Aufgabe Schleifen

Wandeln Sie folgende Schleife in die zwei anderen Ihnen bekannten
Schleifentypen um. Die Funktionalität soll dabei selbstverständlich
erhalten bleiben.

```Java
int n = 10;
while (n >= 0 ) {
  System.out.println(n);
  --n;
}
```

## Aufgabe Ratespiel

a) Schreiben Sie ein Programm das sich eine zufällige Zahl zwischen (inklusiv) 1 und 100 auswählt. Anschließend soll der User seine Schätzung eingeben. Ist die Schätzung richtig beendet das Programm. Ansonsten soll dem User ein Tipp ausgegeben werden. Entweder ist die gesuchte Zahl kleiner oder größer als die geschätzte Zahl. Fragen Sie den User so lange bis er die richtige Zahl erraten hat.

b) Begrenzen Sie die Anzahl der Rateversuche des Users auf 10.
